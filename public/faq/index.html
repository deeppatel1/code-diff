Diff Please FAQ

Instant code diff and code compare with a fast, privacy-first diff checker.

Is Diff Please a code diff checker?
Yes. It is a fast code diff and code compare tool with inline or side-by-side views.
Can I compare code side-by-side or inline?
Yes. Toggle between side-by-side and unified views for any language.
Does it support JSON and YAML tools?
Yes. You can beautify, sort, minify, and convert JSON and YAML.
Is my code uploaded anywhere?
No. Diff Please runs in the browser without sign-in or uploads.
